## 📖 Overview

Sentiment Analyzer Bot is an intelligent chatbot that combines **Natural Language Processing (NLP)** with a clean web interface to understand the emotional tone of user messages and respond accordingly.

Instead of replying with generic responses, the chatbot first determines whether a message expresses a **positive**, **negative**, or **neutral** sentiment, then generates responses that better fit the user's emotional state.

This project demonstrates the integration of **AI**, **Python backend development**, and **modern web technologies** into a complete interactive application.

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/AreebaAliSiddiqui/SentimentAnalyzerBot.git
```

### 2. Navigate into the project

```bash
cd SentimentAnalyzerBot
```

### 3. Create a virtual environment

Windows

```bash
python -m venv .venv
```

Activate

```bash
.venv\Scripts\activate
```

Mac/Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```

## 💡 How It Works

1. The user enters a message.

2. The message is sent to the Flask backend.

3. The sentiment analysis module determines the emotional tone.

4. The chatbot selects an appropriate response.

5. The response is displayed instantly in the web interface.
